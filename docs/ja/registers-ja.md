# レジスタ

> 根拠：一次資料 p.66–p.67

## レジスタ一覧

SC62015 は以下のレジスタを持ちます。

| 記号 | OPコード値 | 2進数 | サイズ | 説明 |
|------|-----------|-------|--------|------|
| A  | 0 | 000 | 8ビット | アキュムレータ（Bレジスタの上位バイトでもある） |
| IL | 1 | 001 | 8ビット | Iレジスタの下位バイト |
| BA | 2 | 010 | 16ビット | B（上位）+ A（下位）の2バイトレジスタ |
| I  | 3 | 011 | 16ビット | ループカウンタ兼汎用（IL + IH） |
| X  | 4 | 100 | 20ビット | インデックスレジスタ（XL + XH + XX） |
| Y  | 5 | 101 | 20ビット | インデックスレジスタ（YL + YH + YX） |
| U  | 6 | 110 | 20ビット | ユーザースタックポインタ（UL + UH + UX） |
| S  | 7 | 111 | 20ビット | システムスタックポインタ（SL + SH + SX） |

## レジスタの分類

命令表では以下の分類記号を使います。

| 記号 | 含まれるレジスタ |
|------|-----------------|
| r  | A, IL, BA, I, X, Y, U, S（全レジスタ） |
| r1 | A, IL（8ビット） |
| r2 | BA, I（16ビット） |
| r3 | X, Y, U, S（20ビット） |
| r4 | X, Y, U, S（r3 と同義） |

## 特記事項

- **IL への書き込み時は IH←0 が同時実行されます**（IH は自動クリア）
- **I レジスタ** はループ命令のカウンタとして使用されます。I=0 のとき 10000h（65536）回のループとみなします
- **F（フラグ）** は PUSHS/POPS/PUSHU/POPU 命令でのみスタック操作の対象になります
- **IMR**（インタラプトマスクレジスタ）は内部 RAM の FBh 番地に相当します

## フラグ

| フラグ | 説明 |
|--------|------|
| Z | ゼロフラグ。演算結果が 0 のとき 1、それ以外は 0 |
| C | キャリーフラグ。キャリーまたはボローが発生したとき 1 |
