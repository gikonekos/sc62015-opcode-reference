# 論理・比較命令

**出典:** *PC-E650/PC-U6000 活用研究*, 工学社, 1995, ISBN 4-87593-206-5 — pp.89–91

## AND opr1, opr2

**動作:** opr1 ← opr1 AND opr2  
**フラグ:** Z: 結果が 0 のときセット / C: 変化なし (p.90)

| opr1 | opr2 | オペコード |
|------|------|------------|
| A | n | `70` n |
| (m) | n | `71` m n |
| [klm] | n | `72` m l k n |
| (n) | A | `73` n |
| (m) | (n) | `76` m n |
| A | (n) | `77` n |

**使用例:**
```
; AND A, 55h  —  A=C7h → A=45h
70 55
```

## XOR opr1, opr2

**動作:** opr1 ← opr1 XOR opr2  
**フラグ:** Z: 結果が 0 のときセット / C: 変化なし (p.90)

| opr1 | opr2 | オペコード |
|------|------|------------|
| A | n | `68` n |
| (m) | n | `69` m n |
| [klm] | n | `6A` m l k n |
| (n) | A | `6B` n |
| (m) | (n) | `6E` m n |
| A | (n) | `6F` n |

> 注意: OR と XOR は AND と同じオペランド形式を持つ。各行のオペコードは、出典の表に記載された先頭値を使用しており、OR は括弧内の第 1 の値、XOR は第 2 の値となる (p.90)。

## TEST opr1, opr2

**動作:** opr1 AND opr2 を計算してフラグに反映するが、結果は破棄する (p.91)  
**フラグ:** Z: 結果が 0 のときセット / C: 変化なし (p.91)

opr1 も opr2 も変更されない。個々のビットを検査するために使用する (p.91)。

| opr1 | opr2 | オペコード |
|------|------|------------|
| A | n | `64` n |
| (m) | n | `65` m n |
| [klm] | n | `66` m l k n |
| (n) | A | `67` n |

**使用例:**
```
; TEST (10h), 55h  —  (10h)=28h, 28h AND 55h = 00h → Z=1
; (内部 RAM 絶対アドレッシング、プレバイト 30h)
65 10 55
```

## CMP opr1, opr2

**動作:** opr1 − opr2 を計算してフラグに反映するが、結果は破棄する (p.89)  
**フラグ:** Z: 結果が 0 のときセット / C: キャリーまたはボローが発生したときセット (p.89)

どちらのオペランドも変更されない。条件分岐命令の直前に使用する (p.89)。

| opr1 | opr2 | オペコード |
|------|------|------------|
| A | n | `60` n |
| (m) | n | `61` m n |
| [klm] | n | `62` m l k n |
| (n) | A | `63` n |

**CMP A, 50h のフラグ結果 (p.89):**

| A の値 | Z | C |
|--------|---|---|
| 00h–4Fh | 0 | 1 |
| 50h | 1 | 0 |
| 51h–FFh | 0 | 0 |
