# 交換命令

**出典:** *PC-E650/PC-U6000 活用研究*, 工学社, 1995, ISBN 4-87593-206-5 — pp.80–81

---

## EX opr1, opr2

**動作:** opr1 ↔ opr2  
**フラグ:** 影響なし (p.80)

opr1 と opr2 の内容を交換する。両オペランドは同じサイズでなければならない (p.80)。

> 注意: EX B, A と EX A, B は同一命令であり、逆順に対する個別のオペコードは存在しない (p.80)。

### ❶ レジスタ ↔ レジスタ

| opr1 | opr2 | オペコード |
|------|------|------------|
| A | B | `DD` |
| r2 | r2' | `ED` r2 r2' |
| r3 | r3' | `ED` r3 r3' |

r2/r3 形式の第 2 バイト: 各レジスタを 4 ビットでエンコード（上位ニブル = opr1、下位ニブル = opr2）。

### ❷ 内部 RAM ↔ 内部 RAM

1、2、または 3 バイトを交換する。絶対アドレッシングにはプレバイトが必要 (p.80)。

| ニーモニック | opr1 | opr2 | オペコード |
|------------|------|------|------------|
| EX | (m) | (n) | `C0` m n |
| EXW | (m) | (n) | `C1` m n |
| EXP | (m) | (n) | `C2` m n |

**使用例:**
```
; EX X, Y  —  X=12345h, Y=6789Ah → X=6789Ah, Y=12345h
; r3=4=X, r3'=5=Y → 第 2 バイト = 45h
ED 45

; EXW (10h), (20h)  —  10h と 20h にある 2 バイトを交換
; (内部 RAM 絶対アドレッシング、プレバイト 32h)
C1 10 20
```

## EXL opr1, opr2 — 交換ループ

**動作:** I レジスタのバイト数分、opr1 と opr2 の内部 RAM 領域を交換し、両アドレスをインクリメントする (p.81)  
**フラグ:** 影響なし (p.81)  
**実行後:** I ← 0

内部 RAM 領域のループ交換。オペランドの組み合わせは内部 RAM ↔ 内部 RAM のみ (p.81)。

> 注意: `MVL (n), [r3]` や `MVL [r3], (n)` が存在しないのと同様に、EXL も内部 RAM 専用である (p.81)。

| opr1 | opr2 | オペコード |
|------|------|------------|
| (m) | (n) | `C3` m n |

**使用例:**
```
; EXL (30h), (40h)  —  3 バイトを交換 (I=0003h)
; (内部 RAM 絶対アドレッシング、プレバイト 32h)
; 実行前: (30h–32h)=12h,34h,56h  (40h–42h)=ABh,CDh,EFh
; 実行後: (30h–32h)=ABh,CDh,EFh  (40h–42h)=12h,34h,56h
; I → 0000h
C3 30 40
```
