# シフト・ローテート命令

**出典:** *PC-E650/PC-U6000 活用研究*, 工学社, 1995, ISBN 4-87593-206-5 — pp.92–94

---

## ROR opr — 右ローテート

**動作:** opr を 1 ビット右にローテートする。ビット 0 がビット 7 および C に移動する (p.92)  
**フラグ:** Z: 結果が 0 のときセット / C: ビット 0 からシフトアウトされた値 (p.92)

```
ROR:  [bit7 ← bit6 ← ... ← bit1 ← bit0] → C
       ↑_______________ (bit0 が bit7 に折り返す) _____|
```

| opr | オペコード |
|-----|------------|
| A | `E4` |
| (n) | `E5` n |

**使用例:**
```
; ROR A  —  A=B5h (1011 0101), C=0
; 実行後:   A=DAh (1101 1010), C=1
E4
```

> 注意: ROR と SHR は異なる。ROR ではビット 0 がビット 7 に折り返す。SHR ではビット 7 に C が入る (p.92)。

---

## ROL opr — 左ローテート

**動作:** opr を 1 ビット左にローテートする。ビット 7 がビット 0 および C に移動する (p.92)  
**フラグ:** Z: 結果が 0 のときセット / C: ビット 7 からシフトアウトされた値 (p.92)

```
ROL:  C ← [bit7 → bit6 → ... → bit1 → bit0]
            |_____ (bit7 が bit0 に折り返す) _______↑
```

| opr | オペコード |
|-----|------------|
| A | `E6` |
| (n) | `E7` n |

---

## SHR opr — 右シフト

**動作:** opr を 1 ビット右にシフトする。C がビット 7 に入る。ビット 0 が C に移動する (p.93)  
**フラグ:** Z: 結果が 0 のときセット / C: ビット 0 からシフトアウトされた値 (p.93)

```
SHR:  C → [bit7 ← bit6 ← ... ← bit1 ← bit0] → C
```

| opr | オペコード |
|-----|------------|
| A | `F4` |
| (n) | `F5` n |

**使用例:**
```
; SHR A  —  A=B5h (1011 0101), C=0
; 実行後:   A=5Ah (0101 1010), C=1
F4
```

> 注意: SHR と ROR は異なる。SHR では C がビット 7 に入る。ROR ではビット 0 が直接ビット 7 に折り返す (p.93)。

---

## SHL opr — 左シフト

**動作:** opr を 1 ビット左にシフトする。C がビット 0 に入る。ビット 7 が C に移動する (p.93)  
**フラグ:** Z: 結果が 0 のときセット / C: ビット 7 からシフトアウトされた値 (p.93)

```
SHL:  C ← [bit7 → bit6 → ... → bit1 → bit0] ← C
```

| opr | オペコード |
|-----|------------|
| A | `F6` |
| (n) | `F7` n |

---

## DSRL opr — 10 進右シフトループ

**動作:** opr から始まる内部 RAM の I レジスタ分のバイトを 4 ビット右にシフトし、アドレスをインクリメントする (p.94)  
**フラグ:** Z: すべてのバイトが 0 になったときセット / C: 変化なし (p.94)  
**実行後:** I ← 0

内部 RAM に格納された BCD 数値を 1 桁 (4 ビット) 右にシフトする。BCD 値を 10 で割る操作に相当する (p.94)。

| opr | オペコード |
|-----|------------|
| (n) | `FC` n |

**使用例:**
```
; DSRL (10h)  —  I=0004h, (10h–13h) = 12h,34h,56h,78h
; 実行後: 01h,23h,45h,67h  (4 ビット右シフト = ÷10)
; (内部 RAM 絶対アドレッシング、プレバイト 30h)
FC 10
```

## DSLL opr — 10 進左シフトループ

**動作:** opr から始まる内部 RAM の I レジスタ分のバイトを 4 ビット左にシフトし、アドレスをデクリメントする (p.94)  
**フラグ:** Z: すべてのバイトが 0 になったときセット / C: 変化なし (p.94)  
**実行後:** I ← 0

BCD 数値を 1 桁 (4 ビット) 左にシフトする。BCD 値を 10 倍する操作に相当する (p.94)。

| opr | オペコード |
|-----|------------|
| (n) | `EC` n |

## SWAP A — ニブル交換

**動作:** A[0–3] ↔ A[4–7]（A の上位 4 ビットと下位 4 ビットを交換する）(p.100)  
**フラグ:** Z: 結果が 0 のときセット / C: 変化なし (p.100)

| オペコード |
|------------|
| `EE` |

**使用例:**
```
; SWAP A  —  A=3Ah → A=A3h
EE
```
