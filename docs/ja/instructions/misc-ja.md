# その他の命令

**出典:** *PC-E650/PC-U6000 活用研究*, 工学社, 1995, ISBN 4-87593-206-5 — pp.100–103

---

## SC — キャリーセット

**動作:** C ← 1  
**フラグ:** Z: 変化なし / C: 1 にセット (p.100)

| オペコード |
|------------|
| `97` |

```
; SC
97
; 実行後: C = 1
```

---

## RC — キャリーリセット

**動作:** C ← 0  
**フラグ:** Z: 変化なし / C: 0 にリセット (p.101)

| オペコード |
|------------|
| `9F` |

```
; RC
9F
; 実行後: C = 0
```

## WAIT — ウェイト

**動作:** CPU が (I + 1) サイクル停止する。毎サイクル I ← I − 1 し、I = 0 で終了 (p.101)  
**フラグ:** 影響なし (p.101)  
**サイクル数:** 1+I

遅延を挿入するために使用する。2.304 MHz 動作時の 1 サイクル = 1.30 μs (p.101)。実行前に I に待機サイクル数をセットしておくこと。

| オペコード |
|------------|
| `EF` |

**使用例:**
```
; 1Fh (31) サイクル待機  (I=001Eh → 001Eh+1 = 1Fh サイクル)
MV  I, 001Eh   ; 0B 1E 00
WAIT           ; EF
; 実行後: I = 0000h
```

## HALT — システムクロック停止

**動作:** システムクロックを停止する。CPU が停止する (p.102)  
**フラグ:** 影響なし (p.102)

割り込みが発生すると実行が再開される (p.102)。

| オペコード |
|------------|
| `DE` |

---

## OFF — システムクロックおよびサブクロック停止

**動作:** システムクロックとサブクロックを停止する (p.102)  
**フラグ:** 影響なし (p.102)  
**サイクル数:** 出典では判読不能 (p.230)

CTRL+OFF キー操作と同等。CPU と LCD コントローラを停止する。電源オンで動作が復帰する (p.102)。

| オペコード |
|------------|
| `DF` |

---

## IR — ソフトウェア割り込み

**動作:** IMR とフラグをシステムスタックに保存し、FFFFFAh (3 バイト) にある割り込みベクタを呼び出す (p.102)  
**フラグ:** 影響なし (p.102)

ソフトウェア割り込みを発生させる。割り込みハンドラへジャンプする前に、IMR (内部 RAM FBh) とフラグがシステムスタックにプッシュされる (p.102)。

| オペコード |
|------------|
| `FE` |

---

## RESET — ソフトウェアリセット

**動作:** リセットベクタ (FFFFFDh、3 バイト) に格納されたアドレスへジャンプする (p.103)  
**フラグ:** 影響なし (p.103)

リセットベクタを読み込んでジャンプすることでソフトウェアリセットを実行する (p.103)。

| オペコード |
|------------|
| `FF` |
