# スタック命令

**出典:** *PC-E650/PC-U6000 活用研究*, 工学社, 1995, ISBN 4-87593-206-5 — pp.82–83

SC62015 には 2 種類のスタックがある:

| スタック | ポインタ | プッシュ方向 | 使用命令 |
|---------|---------|------------|---------|
| システムスタック | S レジスタ | S がデクリメント | CALL/CALLF/RETI |
| ユーザースタック | U レジスタ | U がデクリメント | PUSHU/POPU |

---

## PUSHS opr — システムスタックへプッシュ

**動作:** `MV [--S], opr` と等価 (p.82)  
**フラグ:** 影響なし（F オペランドについては POPS を参照）

opr をシステムスタック（S ベース）に保存する。対応する MV 命令と同等である (p.82)。

| opr | オペコード |
|-----|------------|
| r1 | `MV r1, [--S]` と等価 |
| r2 | `MV r2, [--S]` と等価 |
| X | `MV X, [--S]` と等価 |
| Y | `MV Y, [--S]` と等価 |
| F | `4F` |
| IMR | `MV (IMR), [--S]` と等価 |

> F および IMR オペランドについては下記の POPS の注記を参照 (p.82)。

---

## POPS opr — システムスタックからポップ

**動作:** `MV opr, [S++]` と等価 (p.82)  
**フラグ:** opr = F のとき Z と C が変化する (p.82)

opr をシステムスタックから復元する。

| opr | オペコード |
|-----|------------|
| r1 | `MV r1, [S++]` と等価 |
| r2 | `MV r2, [S++]` と等価 |
| X | `MV X, [S++]` と等価 |
| Y | `MV Y, [S++]` と等価 |
| F | `5F` |
| IMR | `MV (IMR), [S++]` と等価 |

**フラグに関する注記:** opr = F のとき、C と Z がスタックの値から復元される。これは POPS がフラグに影響する唯一のケースである (p.82)。

---

## PUSHU opr — ユーザースタックへプッシュ

**動作:** U ← U - size(opr)、その後 [U] ← opr。`MV [--U], opr` と等価 (p.83)  
**フラグ:** 影響なし (p.83)

opr をユーザースタック（U ベース）に保存する (p.83)。

| opr | オペコード |
|-----|------------|
| r1 (A, IL) | `(28+r1)` |
| r2 (BA, I) | `(28+r2)` |
| X | `2C` |
| Y | `2D` |
| F | `2E` |
| IMR | `2F` |

**使用例:**
```
; PUSHU X  —  MV [--U], X と等価
2C

; PUSHU F  —  フラグをユーザースタックに保存
2E
```

---

## POPU opr — ユーザースタックからポップ

**動作:** opr ← [U]、その後 U ← U + size(opr)。`MV opr, [U++]` と等価 (p.83)  
**フラグ:** opr = F のとき Z と C が変化する (p.83)

opr をユーザースタックから復元する (p.83)。

| opr | オペコード |
|-----|------------|
| r1 (A, IL) | `(38+r1)` |
| r2 (BA, I) | `(38+r2)` |
| X | `3C` |
| Y | `3D` |
| F | `3E` |
| IMR | `3F` |

**フラグに関する注記:** opr = F のとき、C と Z がポップした値から更新される (p.83)。

**使用例:**
```
; POPU X  —  MV X, [U++] と等価
3C
```
