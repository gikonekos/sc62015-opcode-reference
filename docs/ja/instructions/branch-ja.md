# 分岐命令

**出典:** *PC-E650/PC-U6000 活用研究*, 工学社, 1995, ISBN 4-87593-206-5 — pp.95–97

---

## JR opr — 相対ジャンプ

**動作:** PC ← PC + 2 ± n  
**フラグ:** 影響なし (p.95)

PC に符号付き 8 ビットオフセットを加算する。オフセットは JR 命令の次の命令アドレスを基準とする（つまりオフセット適用時、PC はすでに 2 進んでいる）(p.95)。

| opr | オペコード |
|-----|------------|
| +n | `12` n |
| -n | `13` n |

**使用例:**
```
; JR +20h — 命令アドレス BDC31h
; 実行時の PC = BDC33h (BDC31h + 2)
; ジャンプ先 = BDC33h + 20h = BDC53h
12 20
```

## JRxx opr — 条件付き相対ジャンプ

**動作:** 条件が真の場合: PC ← PC + 2 ± n。偽の場合: 何もしない。  
**フラグ:** 影響なし (p.95)

| xx | 条件 | オペコード |
|----|------|------------|
| Z | Z = 1 | `18` (+n) / `19` (-n) |
| NZ | Z = 0 | `1A` (+n) / `1B` (-n) |
| C | C = 1 | `1C` (+n) / `1D` (-n) |
| NC | C = 0 | `1E` (+n) / `1F` (-n) |

**使用例:**
```
; JRZ +20h — 命令アドレス D0000h、Z=1
; ジャンプ先 = D0022h
18 20

; CMP と組み合わせた典型的な使い方 (p.95):
CMP  A, 20h   ; 60 20
JRC  L1       ; 1C ...   (A < 20h のとき、C=1 でジャンプ)
JRZ  L2       ; 18 ...   (A = 20h のとき、Z=1 でジャンプ)
; A > 20h のときはそのまま続行
```

## JP mn — ショートジャンプ (16 ビット)

**動作:** PC ← mn  
**フラグ:** 影響なし (p.96)

指定した 16 ビットアドレスへジャンプする。PS は変化しないため、ジャンプ範囲は現在の PS セグメント内の 64K バイトに限られる (p.96)。

| opr | オペコード |
|-----|------------|
| mn | `02` n m |

**使用例:**
```
; JP 3A7Dh — アドレス (PS, 3A7Dh) へジャンプ
02 7D 3A
```

---

## JPxx mn — 条件付きショートジャンプ (16 ビット)

**動作:** 条件が真の場合: PC ← mn。偽の場合: 何もしない。  
**フラグ:** 影響なし (p.96)

| xx | 条件 | オペコード |
|----|------|------------|
| Z | Z = 1 | `14` n m |
| NZ | Z = 0 | `15` n m |
| C | C = 1 | `16` n m |
| NC | C = 0 | `17` n m |

**使用例:**
```
; JPNC 1A54h — C=1: 次の命令へ続行
;            — C=0: (PS, 1A54h) へジャンプ
17 54 1A
```

## JPF lmn — ファージャンプ (20 ビット)

**動作:** PC ← lmn の下位 16 ビット、PS ← lmn の上位 4 ビット  
**フラグ:** 影響なし (p.97)

完全な 20 ビットアドレスへジャンプする。PC と PS の両方が更新される。アドレス空間: 1M バイト (p.97)。

| opr | オペコード |
|-----|------------|
| lmn | `03` n m l |

**使用例:**
```
; JPF 0BD12Ah — BD12Ah へジャンプ
03 2A D1 0B
```

---

## JP (n) — 内部 RAM 経由ファージャンプ (20 ビット)

**動作:** PC ← (n) 上位 8 ビット; PC ← (n+1) 下位 8 ビット; PS ← (n+2) の下位 4 ビット  
**フラグ:** 影響なし (p.97)

間接ファージャンプ。アドレス n から始まる内部 RAM の連続 3 バイトから 20 ビットのジャンプ先アドレスを読み取る (p.97)。

| opr | オペコード |
|-----|------------|
| (n) | `10` n |

---

## JP r3 — レジスタ経由ファージャンプ (20 ビット)

**動作:** PC ← r3 の下位 16 ビット、PS ← r3 の上位 4 ビット  
**フラグ:** 影響なし (p.97)

20 ビットレジスタ (X、Y、U、S のいずれか) を使った間接ファージャンプ (p.97)。

| opr | オペコード (第 2 バイト) |
|-----|--------------------------|
| r3 | `11` `0r3` |

**使用例:**
```
; JP X — X = C35BAh → C35BAh へジャンプ
11 04   ; r3=4=X
```
