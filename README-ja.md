# SC62015 Opcode Reference

**SC62015** の opcode・レジスタ・アドレッシング方式を再構築したリファレンス用リポジトリです。

**SC62015** は、SHARP の **PC-E500** 系などで使われている CPU です。  
このリポジトリは、歴史的な印刷資料をもとに再構築した命令表・OPコード表を、参照しやすい形で整理・保存することを目的としています。

---

## 目的

このリポジトリでは、SC62015 命令体系に関する再構築済みの参照情報を整理します。

主な対象は以下です。

- OPコード表
- レジスタ分類
- アドレッシング方式
- 命令概要
- 必要に応じた検証メモ

目的は、解析結果を簡潔で再利用しやすい形で残すことです。

---

## 範囲

このリポジトリは、**最終的な再構築結果の整理公開** に重点を置いています。

以下のような中間作業ファイルは、通常はこのリポジトリには含めません。

- 生スキャン画像
- 一時的な OCR テキスト
- 比較途中の下書きファイル
- 段階的な作業メモ
- 一時検証用の scratch ファイル

これらは解析時には有用でしたが、通常の参照用途には必須ではありません。

---

## 根拠資料について

再構築は、SC62015 命令表に関する歴史的な印刷資料をもとに行っています。

ただし、印刷資料の読取りと手作業による再構築であるため、以下の点に注意してください。

- 転記ミスや解釈ミスが残る可能性があります
- 原資料の状態により、一部判読が難しい箇所があります
- このリポジトリは再構築・参照用であり、公式ベンダ資料そのものではありません

不確かな箇所がある場合は、無理に断定せず、その旨を明記する方針です。

---

## 収録内容

このリポジトリには、例えば以下のようなファイルを置くことを想定しています。

- 再構築した OPコード表
- レジスタ対応表
- アドレッシング方式の説明
- 命令リファレンステキスト
- 英語版・日本語版 README

基本的には、**最終参照用のファイルのみ** を収録します。

---

## 編集方針

このリポジトリでは、次の方針を重視します。

- 不明な情報を作らない
- 読みに自信のない原文を勝手に正規化しない
- 読めないものは読めないと明記する
- 整理後の公開構成はなるべく安定させる

過度に広げるより、明快で追跡可能な形を優先します。

より良い根拠に基づく修正提案があれば歓迎します。

## 著作権と利用について

原資料の著作権状態には十分配慮してください。

このリポジトリは、**再構築した参照情報** の保存を主目的としています。  
将来的に原ページ画像や引用を含める場合は、資料保存・研究の範囲で適切に扱う必要があります。
