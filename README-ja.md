# SC62015 Opcode Reference

原典の印刷資料を検証しながら再構築した **SC62015** のオペコード、レジスタ、アドレッシング方式のリファレンス用リポジトリです。

## 現在の状態

このリポジトリは現在準備中です。

最終的なオペコード表は **まだ公開していません**。  
全件確認が完了した段階で追加します。

## 対象

このプロジェクトでは、主に以下を整理します。

- オペコード表
- レジスタ分類
- 内部RAMアドレッシング
- 外部メモリアドレッシング
- 命令形式とオペランド表記
- 再構築方法に関する注記

## 資料方針

このリポジトリは、以下の厳格な方針で作成します。

- 一次資料の表記をできるだけ尊重する
- 憶測による独自表記を作らない
- 読めない箇所は推定せず、未解決として扱う
- 作業メモと最終確定表はできるだけ分離する

## 予定している内容

- `README.md` — 英語概要
- `README-ja.md` — 日本語概要
- `docs/methodology.md` — 再構築方針と方法
- `docs/ja/methodology-ja.md` — 再構築方針と方法（日本語）
- オペコード参照ファイル — 検証完了後に追加予定

## 注記

このリポジトリは、途中経過をすべて保存することよりも、**整理された最終リファレンスを公開すること** を重視しています。

一部の途中作業は、一時的なスキャン確認環境に依存していたため、公開リポジトリには含めない方針です。

## ライセンス

特記がない限り、このリポジトリは原典の歴史資料そのものを自由コンテンツとして再配布するものではありません。

このリポジトリ内で新たに作成した説明文・注記・整理テキストに対してのみ、リポジトリのライセンスを適用します。
